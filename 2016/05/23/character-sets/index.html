Character Sets

HTML Encoding(Character Sets)

为了正确显示Html网页，需要指定使用的字符集(character set / character encoding)。

  • ASCII
  • ANSI
    ANSI(Windows-1252)是Windows默认的字符集，因此所有的浏览器都支持。
    其控制字符(0-31、127)为控制硬件设备而设计，其在HTML文件中不起作用。
  • ISO-8859-1
    HTML4默认的字符编码为ISO-8859-1。
    当浏览器探测到网页为ISO-8859-1时，通常将其转化为ANSI，因为它俩挺相近哒，只有32个（128-159）character不同，具体不同瞅这里的Important Note部分。这一条在HTML5、XHTML也试用哦。
  • UTF-8(Unicode)
    HTML5默认的字符编码为UTF-8(Unicode)
