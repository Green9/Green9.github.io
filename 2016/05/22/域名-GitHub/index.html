域名+GitHub

买个域名

  • namecheap,第一年有优惠，后面也是贵贵的，ヽ（≧□≦）ノ
  • 阿里云，为什么跟namecheap的价钱一样贵呢呢，都没有优势了。
  • freenom，提供了几个免费的.tk、.ml、.ga、.cf、.gq。

配置GitHub

添加CNAME文件

在repository中添加名为CNAME文件，添加你的域名，比如Green9.xyz。
注意不需要http://或者https://哦。也不要添加多个域名啦，拒绝土豪 ⁄(⁄ ⁄•⁄ω⁄•⁄ ⁄)⁄

更多瞅瞅GitHub Help

配置namecheap

namecheap的网页漂亮些，所以就在那买啦。

  • 登陆后找到domain list，点击购买的域名的manage按钮
  • 在Advanced DNS中添加新的RECORD
    1. A record for @ pointing to 192.30.252.153
    2. A record for @ pointing to 192.30.252.154
    3. CNAME record for www pointing to Green9.github.io
  • 记着保存哦

详细的文档很详细，一下子就配好啦，(•̀ᴗ•́)و ̑̑

问题

  • Pages brance 为master时，文档中写道If your current branch is master, you should choose to create a new branch for your commit and then create a pull request。然而在下一步确认时出现Your site is published at http://green9.github.io,最后并不能成功。而将CNAME文件直接提交到master分支则再次提交时，文件丢失。
    因此我将CNAME文件写在了source目录下。
